# Xamarin Dictionary and Test #

This is a Xamarin Mobile Applicaton which works on all platforms iOS, Android and Windows.

You can Sign UP or Login to the application and create your own dictionary by choosing from the pre-listed options like Enligh to French or French to Dutch, etc. 

You can add your words list and see whats that word called in the other language. 

You can create a test for yourself and see how much you learned the language.


## Dev Env: ##

- Xamarin Forms



## Platform: ##

- iOS
- Android
